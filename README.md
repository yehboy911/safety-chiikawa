# 🐹 兒童版：台灣全民安全指引學習機

這是一款專為國小三年級學生設計的互動學習網頁，主題為《台灣全民安全指引》。
採用 **吉伊卡哇 (Chiikawa)** 的可愛視覺風格，透過背誦與測驗模式，讓防災知識變得更有趣！

## ✨ 特色功能

* **單一檔案運作**：無需安裝任何伺服器或環境，點開 HTML 即可使用。
* **雙模式學習**：
    * 📖 **背誦模式**：完整圖卡與語音朗讀。
    * ✨ **測驗模式**：互動式問答，搭配虛擬數字鍵盤。
* **語音朗讀 (TTS)**：支援台灣口音中文朗讀 (需依裝置設定)。
* **正向回饋**：答對時會有可愛音效與鼓勵語（如：好厲害、一百分）。
* **RWD 響應式設計**：完美支援 iPad 與 iPhone 直式操作。

## 🚀 如何使用 (Quick Start)

1.  下載本專案中的 `safety-chiikawa.html` 檔案。
2.  **電腦使用者**：直接雙擊檔案，使用 Chrome 或 Edge 瀏覽器開啟。
3.  **iPad/iPhone 使用者**：
    * 將檔案傳送到裝置 (透過 AirDrop 或 iCloud)。
    * 使用「檔案」App 開啟，或上傳至 GitHub Pages 透過連結瀏覽。
    * *建議開啟靜音開關旁邊的「鈴聲」以聽見語音。*

## 🛠️ 如何修改題目

如果您想更換題目，請使用文字編輯器 (如 VS Code, Notepad++) 開啟 `html` 檔案：

1.  搜尋關鍵字 `const DATA =`。
2.  依照格式修改內容：
    ```javascript
    {
      id: 1,
      type: "number",
      q: "題目內容",
      a: "答案數字",
      unit: "單位",
      context: "補充說明的文字"
    }
    ```

## 📦 技術棧

* **核心**：HTML5, Native JavaScript
* **框架**：React 18 (透過 CDN 引入)
* **樣式**：Tailwind CSS (透過 CDN 引入)
* **字體**：Zen Maru Gothic (Google Fonts)
* **語音**：Web Speech API

## ⚠️ 注意事項

* 本專案使用 CDN 載入函式庫，**使用時請保持網路連線**。
* 語音品質取決於您的裝置，建議使用 iOS/iPadOS 以獲得最佳體驗 (美佳/Siri 語音)。
